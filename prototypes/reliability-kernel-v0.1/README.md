# AI OS Reliability Kernel v0.1 Prototype

CLI-first prototype for AI OS as an **Agent Reliability Layer**.

## Flow

```txt
Intent → Identity → Permission → Risk Score → Governed Tool → Validation → Cost → Audit Report
```

## Run

```bash
npm install
npm run dev
```

## Safety

- mock tools only
- no external API calls
- no real email sending
- no destructive actions
- no autonomous background execution
